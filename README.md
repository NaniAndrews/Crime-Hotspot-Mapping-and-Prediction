# Crime Hotspot Mapping and Prediction

This project was developed as part of my M.Tech in Remote Sensing at Andhra University.

## Objective
To identify crime hotspots and predict future crime-prone areas using GIS and Machine Learning techniques.

## Tools Used
- ArcGIS Pro
- QGIS
- Python
- Remote Sensing Data

## Methods
- Geocoding
- Kernel Density Estimation (KDE)
- Getis-Ord Gi*
- Spatial Analysis
- Machine Learning Models

## Study Area
Visakhapatnam, Andhra Pradesh, India
